Waa maxay HTML ?

HTML waa luqad loo yaqaan “Markup” oo loo isticmaalo in lagu dhiso Bogaga internetka and Websityada

  • HTML waxay u taagan tahay “Hyper Text Markup Language”
  • Luqada HTML waxay qeexdaa qaabka loo dhiso websityada
  • Astaamaha HTML asaaska salka ah ee lagu dhiso websityada
  • Astaamaha HTML waxaa u taagan waxan afka qalaad loo yaqaan “tags” sida head,body, paragraph ama title iyo wixii la mid ah
  • HTML “tags” waxay calaameeyaan oo qeexaan qeybaha kala duwan ee bogga sid kor ku xusan
  • Browserada oo ah aalad loo adeegsado in lagu galo websityada ma muujiyaan calaamadaha loo yaqaan “tags”, browseradau waa chrome,firefox iyo internet explorer
Xagan ku dhufo si aad u aragto Tusaale af Somali ah oo qeexaya Astaamaha HTML

Qeexitaanka Astaamaha HTML

  1. html => furitaanka iyo qeexitaanka HTML
  2. head => madaxa booga/websiteka ama meesha maculamdka websiteka haysa
  3. title => cinwaanka websiteka
  4. body => Qaybta Muuqata ee websiteka
  5. p => fur cinwaanka paragrafka
  6. h1 => Ciwaanka koowad ee websiteka
  7. img => sawirka websiteka
